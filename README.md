# A02-knn07001-erc25001
A02 Ping Pong Project
This repository contains the work collaborated on by Kevin Nelson and Farhad Farzam for the Ping Pong Project for OPIM 5512. 
This project trains a neural network regression model using MLPRegressor to predict California housing prices. The workflow includes loading the dataset, splitting into training and test sets, fitting the model, and evaluating performance through visualizations. The repository generates three key plots:

Training predictions vs actual values

Test predictions vs actual values

Validation score vs epoch during training

All figures are saved in the figs/ directory.
The project follows a structured Git workflow using feature branches and pull requests to document incremental development.
